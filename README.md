# Selenium Automation Project

Learning selenium webdriver with python. using the-internet.herokuapp.com 
as a practice site since its free and made for this.

## what this does
- automates login, forms and navigation tests
- uses page object model pattern (POM)
- 22 tests total

## how to run
pip install -r requirements.txt
pytest -v

## stuff i learned
- how WebDriverWait works vs time.sleep
- page object model pattern
- pytest fixtures (conftest.py)
